Openapi + Spring-boot + Swagger-ui
==================================

Using maven with pom.xml
------------------------

Start jetty
- gmake mvnRun

Build jar then start jetty with this jar
- gmake mvnRunWithJar



Using gradle with build.gradle
------------------------

Start jetty
- gmake gradleRun

Build jar then start jetty with this jar
- gmake gradleRunWithJar
